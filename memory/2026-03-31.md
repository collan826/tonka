## 🔧 后端 API 接口（已实现）

### 认证相关
- `POST /api/auth/login` - 登录
- `GET /api/auth/info` - 获取管理员信息
- `POST /api/auth/logout` - 退出登录

### 公开 API
- `GET /api/public/config` - 获取系统配置
- `GET /api/public/banners` - 获取轮播图列表
- `GET /api/public/car-service` - 获取汽车服务
- `GET /api/public/car-service/points` - 获取汽车服务热点按钮
- `GET /api/public/professional-services` - 获取专业服务
- `GET /api/public/professional-accessories` - 获取专业配件
- `GET /api/public/hot-products` - 获取热门产品

### 文件上传
- `POST /api/file/upload` - 上传图片
